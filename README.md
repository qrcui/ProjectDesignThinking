# VisionGuard AI

Privacy-first visual-acuity screening and digital-eye-strain monitoring across the web, Android, Windows, Linux, and macOS.

注重隐私的视力筛查与数字视疲劳监测应用，支持网页、Android、Windows、Linux 与 macOS。

注重隱私的視力篩查與數位視覺疲勞監測應用程式，支援網頁、Android、Windows、Linux 與 macOS。
